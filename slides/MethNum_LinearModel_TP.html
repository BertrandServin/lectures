Comparison of means : Tukey HSD

  1. Using the emmeans package, and the emmeans function of this package, compute all pairwise differences between mean treatments using Tukey’s adjustment for multiple comparisons emmeans(<anova object>, pairwise ~ <cofactor>, adjust = "tukey" )
  2. using the cld function of the multcomp package, calculate the Compact Letter Representation for the difference in means between treatments on the response.
Code 
boxplot(decrease ~ treatment, data=OrchardSprays)
Code 
aov.orch = lm(decrease ~treatment, data=OrchardSprays)
aov.means = emmeans(aov.orch, pairwise ~ "treatment", adjust="tukey")
aov.means.cld = cld(aov.means$emmeans, Letters = letters)
aov.means.cld
